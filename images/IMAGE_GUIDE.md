## üí° Tips

1. **File Names:** Use exact file names as shown (case-sensitive)
2. **Image Quality:** Optimize images for web (compress but maintain quality)
3. **Formats:** JPG for photos, PNG for logos with transparency
4. **Sizes:** Larger images work, but optimize file size for faster loading

## üìù Notes

- All file names are **case-sensitive**
- Use **lowercase** letters for file names
- Use **hyphens (-)** instead of spaces in file names
- Ensure images are in the correct subfolders (`rooms/` and `reviews/`)
