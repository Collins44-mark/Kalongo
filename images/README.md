# Images Folder

Please add the following images to this folder:

## Logo
- **File:** `logo.png`
- **Location:** `images/logo.png`
- **Recommended size:** 80x80 pixels or larger (square aspect ratio)
- **Format:** PNG with transparent background preferred

## Hero Background
- **File:** `hero-background.jpg`
- **Location:** `images/hero-background.jpg`
- **Recommended size:** 1920x1080 pixels or larger
- **Format:** JPG or PNG

## Room Images
Place room images in the `rooms/` subfolder:

- **A-Cabin:** `images/rooms/a-cabin.jpg`
- **Cottage:** `images/rooms/cottage.jpg`
- **Kikota:** `images/rooms/kikota.jpg`

**Recommended size:** 800x600 pixels or larger
**Format:** JPG or PNG

## Notes
- Make sure all image files match the exact filenames listed above
- Images should be optimized for web (compressed but good quality)
- Supported formats: JPG, PNG, WebP
