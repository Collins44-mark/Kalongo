# Review Images

Please add customer review images to this folder:

1. **customer1.jpg** - Customer review image for slide 1
2. **customer2.jpg** - Customer review image for slide 2
3. **customer3.jpg** - Customer review image for slide 3

## Image Specifications:

- **Recommended size:** 400x400 pixels (square) or 500x600 pixels (portrait)
- **Format:** JPG or PNG
- **Quality:** High quality, optimized for web
- **Content:** Photos of customers, guests at the farm, or happy families enjoying their stay

## Notes:

- All images should be optimized for web viewing
- Supported formats: JPG, PNG
- Images will be displayed in a circular/rounded container
- Make sure images are clear and represent positive guest experiences
